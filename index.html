Basics of the Web

HTML, URLs, HTTP, Web Apps

WWW: a collection of HTML docs (basis of the web)
HTML: stands for Hypertext Markup Languages (glues everything together)
Hyperlinks: glue pages together, making them "weblike"

Major Pieces of the Web:
three components: you-->the internet<--HTTP-->servers,
......where "you"consists of yourself (!), a computer and a browser

HTML:
made up of text content (what you see), mark ups (what it looks like)
and references to other documents (images, videos, etc.)

Sample HTML Markups:
- bold: <b>contents </b>
- italics: <em>contents </em>
- link: <TAG ATTR="value">contents </TAG>
- TAG anchor: <a></a>
......sooo....to create a link: <a href="www.reddit.com">derp</a>
- images: <img> ...and.... <img src="url" alt="text">
......because...it is nice if the image does not display text indicates what the image
might have been. this is nice for peeps who are blind or searches (I think) :)
......also, images don't have closing b/c they are void tags with no content (text)

White Space:
-break: <br> tag or void tag
-paragraph: <p>contents</p>

Inline v. Block Elements:
why two?
<br> tag is an inline element (end of a line)
<p> tag is a block element
......a sort of "box" is created around the text (sort of b/c it is sort of...invisible!)
